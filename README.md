# Tic-Tac-Toe (Rust + Leptos)

A simple Tic-Tac-Toe game built with [Rust](https://www.rust-lang.org/), [Leptos](https://leptos.dev/), and [TailwindCSS](https://tailwindcss.com/).

## Prerequisites

- Rust (`cargo`)
- Trunk (`cargo install trunk`)

## Running Locally

1. Clone the repo.
2. Run the development server:
   ```bash
   trunk serve
   ```
3. Open http://127.0.0.1:8080
